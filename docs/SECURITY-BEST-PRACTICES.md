# Security Best Practices

## For Users
- Verify host fingerprints on first connection.
- Do not disable strict host key checking.
- Use SSH keys instead of passwords where possible.
- Rotate keys and passwords regularly.
- Keep VS Code and this extension updated.

## For Projects
- Keep SSH server algorithms up to date (disable weak ciphers).
- Restrict server access by IP and user.
- Limit permissions of the deployment user.
